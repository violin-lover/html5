Triangles

When calculating for the hypotnuse of a triangle, use the Pythagorean Theorem: \[a^2 + b^2 = c^2\]

To calculate the area of a triangle, use this formula: \[b * h * \frac{1}{2}\]

Circles

To calculate the area of a circle, you need this formula: \[A = \pi*r^2\]

To calculate the circumference of a circle, you need this formula: \[2 * \pi * r\]

COOL FORMULA: to calculate the volume of a cylinder, you need to use the formula: \[V = \pi*r^2*h\]

Squares and Rectangles

Area

To calculate the area of a square, you can use the formula: \[l^2--or--w^2\]

To calculate the area of a rectangle, you need to use this formula: \[2 * l * w\]

KEY!!! ==> When calculating the area of a square, you can use the formula for the area of a rectangle too because it is also a rectangle.

Perimeter

To calculate the perimeter of ANY polygon, just add up the lengths of all the sides. For example, if a rectangle had a length of 4 and a width of 2, you would use this formula: \[2 + 4 + 2 + 4 = 12\]

Speed, Distance, and Time

Speed

To calculate speed, use the formula: \[s = \frac{d}{t}\]

Distance

To calculate distance, use the formula: \[d = s*t\]

Time

To calculate time, use the formula: \[t = \frac{d}{s}\]
